## 三、如何理解回溯法
抽象为树型结构
>树的宽度：集合的大小，用for
> 树的深度：递归的深度
> 
## 四、回溯模版
```
void backtracking ( 参数 ) {
    if (终止条件)
        收集结果，
        return
    
    for (选择：本层集合中元素（树中节点孩子的数量就是集合的大小）) {
        处理节点;
        backtracking(路径，选择列表); // 递归
        回溯，撤销处理结果
    }
}
```
