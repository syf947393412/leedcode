## 思路
### 1.DP数组以及下际的含义
dp[i][j]:以i-1为结尾的s中有以j-1为结尾的t的个数  

### 2.递推公式
```python
if s[i-1]=s[j-1]:
    dp[i][j]=dp[i-1][j-1]+dp[i-1][j]
else:
    dp[i][j]=dp[i-1][j]
```
![img_3.png](img_3.png)
### 3.DP数组如何初始化
dp[0][j]=0  
dp[i][0]=1  
dp[0][0]=1  

### 4.遍历顺序
![img_2.png](img_2.png)
```python
for i in range(1,len(s)+1)
    for j in range(1,len(t)+1)
```
