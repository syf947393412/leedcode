## 思路
### 1.DP数组以及下际的含义
dp[i][j]:以[0,i-1]为区间的text1 和 以[0,j-1]为区间的text2的最长公共子序列长度为dp[i][j]  

### 2.递推公式
```python
if text1[i-1]==text2[j-1]:
    dp[i][j]=dp[i-1][j-1]
else:
    dp[i][j]=max(dp[i-1][j],dp[i][j-1])

```

### 3.DP数组如何初始化
dp[i][0]=dp[0][j]=0
### 4.遍历顺序
```python
for i in range(1,len(text1)+1):
    for j in range(1,len(text2)+1):
```
