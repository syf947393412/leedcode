## 思路
### 1.DP数组以及下际的含义
dp[i][j]:[i,j] 的回文子序列的长度为dp[i][j]
### 2.递推公式
```python
if s[i]==s[j]:
    dp[i][j]=dp[i+1][j-1]+2
else:
    dp[i][j]=max(dp[i][j-1],dp[i+1][j])
```
![img_4.png](img_4.png)
### 3.DP数组如何初始化
`dp[i][i]=1`(i和j相等时，回文长度是1)
```python
for i in range(len(s)):
    dp[i][i]=1
```
### 4.遍历顺序
- 从下往上，从左往右 
![img_5.png](img_5.png)
```python
for i in range(len(s)-1,-1,-1):
    #i=j时已经初始化过了，所以j从i+1开始
    for j in range(i+1,len(s),1):
```
- 求：`dp[0][len(s)-1]`
