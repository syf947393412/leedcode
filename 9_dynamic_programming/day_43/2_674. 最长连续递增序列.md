## 思路
### 1.DP数组以及下际的含义
dp[i]:以i为结尾的最长连续递增序列的长度

### 2.递推公式
```python
if nums[i]>nums[i-1]:
    dp=dp[i-1]+1
```
### 3.DP数组如何初始化
dp[0]=1
### 4.遍历顺序
- 正序
